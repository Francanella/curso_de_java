TEMA 1: Características de Java
- Características de Java
-- Alto nivel y uso general
    Es un lenguaje de programación bastante amigable con el
    programador, es decir, su sintaxis es relativamente
    cómoda para interpretar por un programador.
    Es de uso general porque tiene aplicaciones muy diversas
    las cuales, a veces, estan atendidas por diferentes
    lenguajes de programación.
-- Orientado a objetos
    Orientado a objetos (POO), es un paradigma de programación
    Una metodología, una forma de programar.
    Se basa en Objetos (que simulan objetos del modelo de dominio
    de la aplicación)
-- Independiente de la plataforma
    Porque esta diseñado para que el programador, una vez escrito
    el programa, se desentienda de la plataforma (sistema 
    operativo) en el cual se ejecutará.
-- Compilado
    Compilado significa que el código que escribe el programador
    (código fuente), es transformado por un programa denominado
    compilador en otro código entendible por el sistema (máquina
    virtual de Java o JVM).
TEMA 2: Aplicaciones de Java
- Aplicaciones de escritorio
    Son aquellas en donde el conjunto de los elementos de la 
    aplicación residen en un solo sistema.
- Aplicaciones de servidor
    Son aplicaciones que dan servicios a través de una red.
- Aplicaciones de cliente
    Son las aplicaciones que consultan servicios a través de
    una red y hacen de interfaz con el usuario.
- Aplicaciones móbiles
    Son las aplicaciones que se ejecutan en un dispositivo móvil
    (por ejemplo, smartphones), normalmente, usando el sistema
    operativo Android.
- Juegos
    Son aplicaciones que sirven para entretenimiento.
